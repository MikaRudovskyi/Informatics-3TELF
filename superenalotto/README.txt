### Program Name: SuperEnalotto Game

### Description:
This program simulates a simplified version of the SuperEnalotto lottery game. The user is prompted to input six unique numbers between 1 and 90. The program then generates six random numbers within the same range and compares them with the user's numbers to determine how many matches there are. The user is given the option to play again after each round.

### Explanation:
1. containsNumber Function: Checks if a given number is already in the array.
2. generateRandomNumbers Function: Generates an array of unique random numbers between 1 and 90.
3. main Function: 
   - Prompts the user to enter six unique numbers.
   - Generates six unique random numbers.
   - Compares the user's numbers with the random numbers to find matches.
   - Displays the user's numbers, the random numbers, and any matches.
   - Asks the user if they want to play again and restarts the game if they agree.
